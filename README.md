# script_sauvegarde.

Le script realise le sauvegarde en créant deux archives tar.gz. En utilisant cette paterne: date_nom-archives.tar. 

La premiere archive contient les repertoires "documents": .mozilla,Destop, Documents, Public, Templates. 
La deuxieme archine contient les rpertoires "médias": Entertaiment.
